# PT Citra Gemilang Prima — Corporate Website

Premium B2B single-page website for CGP, Indonesia's trusted gateway for imported food brands.

**Live:** https://jkurn.github.io/cgp-website/

## Contents

- Hero with full-width background & CTAs
- Stats band (20+ years, 12 brands, 6 retail partners)
- Company profile with split-image layout
- Our Values — 3 animated cards
- Brand Principals carousel — Kanokwan, Simplot (Leggo's / Edgell / John West), Snackbrands
- Portfolio section — Brand & Category tabs with modal detail drawers
- Distribution network marquee
- Contact section — WhatsApp CTA + B2B inquiry form
